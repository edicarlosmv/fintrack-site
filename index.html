Política de Privacidade

Última atualização: junho de 2026

O FinTrack é um aplicativo de controle financeiro pessoal. Esta Política descreve como coletamos, usamos e protegemos suas informações.

1. Dados coletados

Coletamos apenas os dados que você mesmo insere:

  • Lançamentos financeiros (descrição, valor, categoria, data)
  • Informações de parcelas (banco, dia de vencimento)
  • Metas financeiras (nome, valor alvo, prazo)
  • Preferência de idioma

Não coletamos nome, CPF, e-mail, localização, fotos, contatos ou dados pessoais identificáveis.

2. Armazenamento

Todos os dados ficam armazenados exclusivamente no seu dispositivo, usando armazenamento seguro e criptografado do Android (SecureStore). Nenhum dado é enviado para servidores externos.

3. Compartilhamento

O FinTrack não compartilha seus dados com terceiros, anunciantes ou qualquer outra entidade.

4. Assinaturas e pagamentos

Pagamentos são processados pelo Google Play. O FinTrack não tem acesso a dados de cartão ou informações bancárias.

5. Exclusão de dados

Desinstale o app para remover permanentemente todos os seus dados do dispositivo.

6. Menores de idade

O FinTrack não é direcionado a menores de 13 anos.

7. Contato

Dúvidas? Entre em contato: suporte@fintrack.app

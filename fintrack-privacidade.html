Política de Privacidade

Última atualização: junho de 2026

O FinTrack é um aplicativo de controle financeiro pessoal desenvolvido para ajudar usuários a organizar receitas, despesas e parcelas. Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações.

1. Dados coletados

O FinTrack coleta e armazena apenas os dados que você mesmo insere no aplicativo:

  • Lançamentos financeiros (descrição, valor, categoria, data)
  • Informações de parcelas (número de parcelas, banco/cartão, dia de vencimento)
  • Metas financeiras (nome, valor alvo, prazo)
  • Preferência de idioma (Português ou Inglês)

Não coletamos nome, CPF, e-mail, localização, contatos, fotos, microfone ou qualquer dado pessoal identificável.

2. Onde os dados são armazenados

Todos os seus dados financeiros são armazenados exclusivamente no seu próprio dispositivo, utilizando o sistema de armazenamento seguro do Android (EncryptedSharedPreferences / SecureStore). Nenhum dado é enviado para servidores externos.

3. Compartilhamento de dados

O FinTrack não compartilha seus dados com terceiros, anunciantes, parceiros ou qualquer outra entidade. Seus dados financeiros são privados e permanecem apenas no seu dispositivo.

4. Assinaturas e pagamentos

O FinTrack oferece planos de assinatura processados pelo Google Play. Os dados de pagamento são gerenciados exclusivamente pelo Google e pela plataforma RevenueCat, seguindo as políticas de privacidade dessas empresas. O FinTrack não tem acesso a dados de cartão de crédito ou informações bancárias.

5. Segurança

Os dados armazenados no dispositivo são protegidos pelo sistema de segurança do Android. Recomendamos manter o dispositivo atualizado e protegido por senha ou biometria para garantir a segurança dos seus dados financeiros.

6. Exclusão de dados

Você pode excluir todos os seus dados a qualquer momento diretamente no aplicativo ou desinstalando o FinTrack. A desinstalação remove permanentemente todos os dados armazenados localmente.

7. Menores de idade

O FinTrack não é direcionado a menores de 13 anos e não coleta conscientemente dados de crianças.

8. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas por meio de uma atualização do aplicativo. O uso continuado do FinTrack após as alterações constitui aceitação da nova política.
